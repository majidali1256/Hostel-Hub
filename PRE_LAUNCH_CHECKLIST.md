# 🚀 SaaS Pre-Launch Checklist

## ✅ Legal & Compliance
- [ ] Privacy Policy page
- [ ] Terms & Conditions (Terms of Service)
- [ ] Cookie consent (especially if targeting EU — GDPR)

## 🔐 Auth & Security
- [x] Signup / login flow tested
- [ ] Email verification working
- [x] Password reset flow
- [x] OAuth (Google, etc.) working if included
- [x] Rate limiting (prevent brute force)

## 💳 Payment
- [ ] Payment flow fully tested (success + failure cases)
- [ ] Subscription lifecycle:
  - [ ] Upgrade
  - [ ] Downgrade
  - [ ] Cancel

## 📊 Analytics & Tracking
- [ ] User Event Tracking
- [ ] Page Tracking

## 📣 Marketing Basics
- [ ] Submit page to Google Search Console
- [ ] Submit on other search engines
- [x] Check for SEO basics

## 💬 Feedback Loop
- [x] Contact / Support Email
- [x] Bug report option
